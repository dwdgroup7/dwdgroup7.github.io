Meeting about video data

2017-03-18

On 18th March, Osama and Lihao have a meeting to integrate some of the data with the video (any action include Arduino will be simulated and replaced with valid code to help to test the data visualization of the data in the video). Simply, this information will be displayed in the video when an event occurs, which is pressing a button that is linked by Arduino to stop the video in processing and show the information.

We wrote a two methods/ functions in Java the first one collected a prepared data and loaded them in Java Lists, where the second received the timing when the cycle pause the video then return the information from the list. The information is a description of the place that the cyclist stopped in. Additionally, we tested this method with a small data, and we found that the methods/ function round up timing even with a small fraction.

Moreover, we wrote an event handler in processing that does that displace a text (or image, however, the image is Ø§eavier makes the video a bit laggy) in the video in case of pausing. At the moment the program event handler works with the left click of the mouse, and we are planning to make it an external button which is linked with Arduino.
