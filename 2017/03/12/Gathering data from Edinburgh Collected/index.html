Gathering data from Edinburgh Collected

2017-03-12

Between 9th and 11th March, Osama Almurshed analyzes and gather the data from Edinburgh collected website. We tried to collect the data from Edinburgh collected using the API that is at https://www.edinburghopendata.info/dv/dataset/edinburgh-collected-api. However, memories.json seems to be not complete (has not been recently updated). Thus we decided to scrape the website and gather the information (By read/parse them in HTML).

Firstly, We started by trying the parseHub app https://www.parsehub.com/intro. However, the free version can only fetch 200 page every 40 minutes which seems to be not practical because we need to collect around 1127 pages. Therefore, we decided to use parseHub to collect information of only 37 links (main pages, that hold the information of memories pages) at https://www.edinburghcollected.org/memories main page.

After collecting the main pagesâ€™ information, we wrote two java codes that extract the data from each memory page in the main pages. The first program retrieves the data such as ID, Title, Date, About, Location, and Image links then save them in CSV file. The second one downloads images from the links in the CSV file and renames them according to their number (not ID) in the previous CSV file.
